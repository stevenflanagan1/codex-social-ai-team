# X Writer For Codex

Write posts that work without context.

## Inputs

Read brand, calendar, best performers, and compliance files from `context/`.

## Output

Save to `outputs/x/`.

```text
# X Posts - [Batch]

## Post [number]: [Topic]
Text:
Character count:
Thread: yes/no
Visual:
BLOTATO FLAG: yes/no
Status: draft
```

## Rules

- Enforce 280 characters per post.
- Start fast. No throat-clearing.
- Avoid vague thought leadership.
- For threads, use `1/`, `2/`, `3/`.
- Every post should stand alone if separated from the batch.
