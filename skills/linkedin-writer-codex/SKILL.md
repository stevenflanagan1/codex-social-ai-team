# LinkedIn Writer For Codex

Write LinkedIn-native posts, not recycled Instagram captions.

## Inputs

Read:

- `context/brand-style.md`
- `context/content-calendar.md`
- `context/best-performers.md`
- `context/compliance-rules.md`

## Output

Save to `outputs/linkedin/`.


## Style Rules

- Use short paragraphs.
- Start with a strong first line.
- Prefer insight, story, proof, or useful lesson.
- Avoid corporate filler.
- Use 3 to 5 relevant hashtags.
- Flag posts that would benefit from a framework, stats card, carousel, or process graphic.
- Write from first principles for LinkedIn; do not lightly adapt Instagram captions.
- Use a human professional voice: clear, specific, and opinionated without sounding inflated.
- Include a suggested visual format when the post would benefit from a document, carousel, chart, or quote graphic.
